# Changelog

## v1.2 (2026-04-16)
- Добавлено восстановление существующих сессий Claude Code
- Чекбокс "Запускать в новой вкладке" на Windows 11
- Добавлено отображение активных сессий с возможностью их закрытия
- Добавлена кнопка "Сброс" для исправления ошибки "No conversation found"
- Добавлен вызов SessionEnd хука при закрытии сессии через лаунчер
- Улучшена стабильность работы с кириллицей в путях (UTF-8)
- Исправлены ошибки при закрытии сессий
- Добавлен чекбокс "Запустить в режиме пропуска разрешений" (--dangerously-skip-permissions)
- Кнопка "Сброс" теперь активируется автоматически при наличии сохранённой сессии
- Добавлен диалог подтверждения с опцией "Больше не спрашивать" для кнопки "Сброс"
- Улучшена обратная связь: статусные сообщения автоматически исчезают через 3 секунды
- Открытие папки проекта по правому клику на кнопке сессии
- Горячая клавиша F5 для перезагрузки лаунчера

## v1.1 (2026-04-09)
- Добавлен `-ExecutionPolicy Bypass` для совместимости с Windows 10
- Добавлена версия скрипта в заголовок окна
- Улучшено позиционирование элементов GUI

## v1.0
- Первая версия
- Выбор папки с историей
- Автозапуск Omniroute
- Запуск Claude Code в выбранной папке
