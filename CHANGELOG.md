# Changelog

## v1.3.2 (2026-04-21)
- Переименован проект: Launch Claude Code → Claude Code Launcher
- Исправлены отступы в интерфейсе (симметричные отступы вокруг разделителя)

## v1.3.1 (2026-04-18)
- Добавлен чекбокс "Загружать прошлую сессию" с возможностью отключения автовосстановления
- Добавлено сохранение состояния чекбокса "Запускать в новой вкладке"
- Чекбокс "Запускать в новой вкладке" теперь видим, но неактивен в Windows 10
- Добавлены всплывающие подсказки для кнопки "Сброс" и чекбокса загрузки сессии
- Исправлена ошибка при работе с диалогом подтверждения сброса сессии

## v1.3 (2026-04-17)
- Добавлено автоматическое восстановление активных сессий при запуске лаунчера
- Добавлено сохранение позиции окна (при завершении перемещения и при закрытии)
- Окно автоматически сдвигается вверх при добавлении сессий, если выходит за панель задач
- Объединены все конфигурационные файлы в один cc_launcher.ini (история, позиция, настройки)
- Добавлен диалог подтверждения для режима пропуска разрешений с опцией "Больше не спрашивать"
- Состояние чекбокса "Запустить в режиме пропуска разрешений" теперь сохраняется между запусками
- Добавлен встроенный обработчик ошибок OnError для автоматического логирования
- Включено подробное логирование всех операций (cc_launcher.log)
- Автоматическая миграция данных из старых файлов (cc_history.txt, cc_position.ini)
- Исправлено переключение на сессии node.exe процессов (поиск родительского окна CMD)
- Оптимизировано сохранение позиции окна (только при реальном перемещении, без таймера)

## v1.2 (2026-04-16)
- Добавлено восстановление существующих сессий Claude Code
- Чекбокс "Запускать в новой вкладке" на Windows 11
- Добавлено отображение активных сессий с возможностью их закрытия
- Добавлена кнопка "Сброс" для исправления ошибки "No conversation found"
- Добавлен вызов SessionEnd хука при закрытии сессии через лаунчер
- Улучшена стабильность работы с кириллицей в путях (UTF-8)
- Исправлены ошибки при закрытии сессий
- Добавлен чекбокс "Запустить в режиме пропуска разрешений" (--dangerously-skip-permissions)
- Кнопка "Сброс" теперь активируется автоматически при наличии сохранённой сессии
- Добавлен диалог подтверждения с опцией "Больше не спрашивать" для кнопки "Сброс"
- Улучшена обратная связь: статусные сообщения автоматически исчезают через 3 секунды
- Открытие папки проекта по правому клику на кнопке сессии
- Горячая клавиша F5 для перезагрузки лаунчера

## v1.1 (2026-04-09)
- Добавлен `-ExecutionPolicy Bypass` для совместимости с Windows 10
- Добавлена версия скрипта в заголовок окна
- Улучшено позиционирование элементов GUI

## v1.0
- Первая версия
- Выбор папки с историей
- Автозапуск Omniroute
