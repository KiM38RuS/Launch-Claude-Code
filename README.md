# Claude Code Launcher<img width="128" alt="Claude free icon 2" src="https://github.com/user-attachments/assets/4d8f4512-1bff-4b90-9743-1aa611752d1a" align="left"/>


Лаунчер для быстрого запуска **Claude Code** в нужной папке проекта с автоматическим запуском **Omniroute**.

<br clear="left"/>

## Первоначальная настройка Claude Code

> [!WARNING]
> Этот метод работает **только при включённом VPN!**

Если Claude Code ещё не установлен, воспользуйтесь инструкциями:

* https://youtu.be/LkP6ocAoQkk – *Valency Labs*
* https://youtu.be/XykwVzBsWsI – *Бывший Скамер*
* https://youtu.be/2k7UHNJBpyY – *Игорь Зуевич*

В этих видео показана установка Node.js, установка Claude Code, настройка Omniroute, подключение бесплатного доступа.

## Возможности

<img src="Assets/Screenshot-v2.0.jpg" width="450" align="right">

**Основное:**
* GUI с историей последних 10 папок
* Автозапуск и проверка Omniroute
* Запуск Claude Code в выбранной папке

**Управление сессиями:**
* Восстановление и отображение активных сессий
* Переключение между сессиями и их закрытие
* Кнопка "Сброс" для исправления ошибки "No conversation found"
* Открытие папки проекта (правый клик на сессии)

**Дополнительно:**
* Запуск в новой вкладке Windows Terminal (Windows 11)
* Режим пропуска разрешений (`--dangerously-skip-permissions`)
* Горячая клавиша F5 для перезагрузки
* Автоисчезающие статусные сообщения
