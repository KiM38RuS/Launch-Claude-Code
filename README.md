# Launch Claude Code<img width="128" alt="Claude free icon 2" src="https://github.com/user-attachments/assets/4d8f4512-1bff-4b90-9743-1aa611752d1a" align="left"/>


Небольшой лаунчер, упрощающий запуск **Claude Code** в нужной папке проекта.
Скрипт позволяет быстро выбрать папку, автоматически запустить **Omniroute** (если он ещё не запущен) и открыть терминал с командой `cc`.

## Первоначальная настройка Claude Code

Если Claude Code ещё не установлен, воспользуйтесь инструкциями:

* https://youtu.be/LkP6ocAoQkk
* https://youtu.be/XykwVzBsWsI

В этих видео показано:

* установка Node.js
* установка Claude Code
* настройка Omniroute
* подключение бесплатного доступа

## Возможности

* GUI для выбора папки проекта
* История последних использованных папок (до 10)
* Автоматический запуск **Omniroute**, если он не запущен
* Проверка успешного запуска Omniroute
* Запуск **Claude Code** в выбранной папке
* Отображение статуса запуска
* Минимальный и быстрый интерфейс

## Как это работает

1. Скрипт открывает окно выбора папки.
2. Пользователь выбирает папку проекта.
3. Скрипт проверяет, запущен ли **Omniroute**.
4. Если нет — запускает его через PowerShell.
5. После этого открывается `cmd` в выбранной папке и выполняется команда `cc`. Это запускает **Claude Code** в текущей папке.

## История папок

Скрипт автоматически сохраняет последние папки проектов в файл `cc_history.txt`

Максимум хранится **10 последних папок**.
